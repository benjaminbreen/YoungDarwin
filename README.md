# Young Darwin

Young Darwin is an open-source historical exploration game created by Benjamin
Breen. Set on Floreana Island (then known as Charles Island) in 1835, it follows
the young Charles Darwin during the voyage of HMS *Beagle*.

The game is intended for educational use, especially in history of science
classrooms, but it is also designed to be engaging and fun on its own.

## Gameplay

Players explore the island, observe wildlife and landscapes, document and
collect specimens, travel between distinct environments, and encounter the
people and ecological changes shaping Floreana. Playable animal perspectives,
including a finch and a tortoise, offer different ways to move through and
understand the world.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000/three`.
